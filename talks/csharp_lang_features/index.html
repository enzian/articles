Tuples

  • Lightweight data structures with multiple fields.
							
public (List list, string message) GetList()
{
	return (new List(),  "error");
}

(List list, string message) = GetList();
(List list, string message) valueTuple = GetList();

Inferred tuple element names

							
var tuple = GetList();
List aList = tuple.list;
string msg = tuple.message;

Discards

    A discard is a write-only variable whose name is _ you can assign all of the values that you intend to discard to the single variable. 
							
(_, string message) = GetList();

Local Functions

  • Function inside a method
  • Instead of private methods
  • Two main use cases
    • Public iterator methods
    • Public async methods

Local Functions

							
public static IEnumerable AlphabetSubset3(char start, char end)
{
    if (start < 'a' || start > 'z')
        throw new ArgumentOutOfRangeException(paramName: nameof(start), message: "start must be a letter");
    if (end < 'a' || end > 'z')
        throw new ArgumentOutOfRangeException(paramName: nameof(end), message: "end must be a letter");

    if (end <= start)
        throw new ArgumentException($"{nameof(end)} must be greater than {nameof(start)}");

    return alphabetSubsetImplementation();

    IEnumerable alphabetSubsetImplementation()
    {
        for (var c = start; c < end; c++)
            yield return c;
    }
}

Local Functions

							
public Task PerformLongRunningWork(string address, int index, string name)
{
    if (string.IsNullOrWhiteSpace(address))
        throw new ArgumentException(message: "An address is required", paramName: nameof(address));
    if (index < 0)
        throw new ArgumentOutOfRangeException(paramName: nameof(index), message: "The index must be non-negative");
    if (string.IsNullOrWhiteSpace(name))
        throw new ArgumentException(message: "You must supply a name", paramName: nameof(name));

    return longRunningWorkImplementation();

    async Task longRunningWorkImplementation()
    {
        var interimResult = await FirstWork(address);
        var secondResult = await SecondStep(index, name);
        return $"The results are {interimResult} and {secondResult}. Enjoy.";
    }
}

Async streams

  • Asynchronously return elements from a IEnumberable.
  • Three properties:
    • It's declared with the async modifier
    • It returns an IAsyncEnumerable.
    • The method contains yield return statements to return successive elements in the asynchronous stream.

Async streams

							
public static async System.Collections.Generic.IAsyncEnumerable GenerateSequence()
{
    for (int i = 0; i < 20; i++)
    {
        await Task.Delay(100);
        yield return i;
    }
}

await foreach (var number in GenerateSequence())
{
    Console.WriteLine(number);
}

Default literal expressions

  • Default literal expressions are an enhancement to default value expressions. These expressions initialize a variable to the default value.
							
Before: 
	Func whereClause = default(Func<string, bool>);
								
After:
	Func whereClause = default;

Readonly members

  • Struct can have readonly members without being readonly itself.
							
public struct Point
{
    public double X { get; set; }
    public double Y { get; set; }
    public readonly double Distance => Math.Sqrt(X * X + Y * Y);

    public readonly override string ToString() =>
        $"({X}, {Y}) is {Distance} from the origin";
}
